# PasswordMaster
Ferramenta em Python que checa se sua senha já vazou em algum lugar (usando a API do Have I Been Pwned, sem nunca expor sua senha de verdade) e avalia se ela é forte de verdade, seguindo o NIST 800-63B. Ela explica cada problema encontrado em linguagem simples, pra quem tá começando em cyber entender sem sofrer com jargão.
